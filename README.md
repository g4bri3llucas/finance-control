# 💰 FinanceControl

Sistema de controle financeiro pessoal construído com React.

## Funcionalidades

- ➕ Cadastro de receitas e despesas
- 📅 Filtro por mês
- 📊 Gráfico mensal comparativo
- ⬇️ Exportação para CSV
- 🗑️ Exclusão de transações

## Diferenciais técnicos

- Lazy loading com Suspense
- Componentização em arquivos separados
- Pipes personalizados para formatação de dados
- Hooks: useState, useMemo

## Tecnologias

- React
- Recharts
- JavaScript (ES6+)

## Como rodar
```bash
npm install
npm start
```
